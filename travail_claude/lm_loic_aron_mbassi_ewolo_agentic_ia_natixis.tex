Madame, Monsieur,

Les architectures agentic -- agents LLM capables de planifier, d'utiliser des outils et d'it\'erer sur des t\^aches complexes -- repr\'esentent la fronti\`ere actuelle de l'ing\'enierie IA. Natixis, en investissant dans le d\'eveloppement d'une application No Code bas\'ee sur des agents g\'en\'eratifs, se positionne sur l'une des transformations les plus profondes de l'automatisation bancaire. Je veux contribuer \`a cette transformation.

Mon outil Laravel API Generator (2024, +30 \'etoiles GitHub) est l'illustration la plus directe de ma ma\^itrise des architectures agentic : un syst\`eme qui lit un diagramme UML, planifie la structure d'une API REST, g\'en\`ere le code PHP via LLM et produit les tests automatis\'es. C'est un pipeline agentic complet, livr\'e en production et adopt\'e par la communaut\'e. Le projet Snappy (2025) int\`egre un module RAG dans une messagerie temps r\'eel, avec recherche s\'emantique et g\'en\'eration de r\'eponses contextualis\'ees. Mon stage INRIA Saclay (avril-ao\^ut 2026) ancre ces comp\'etences dans un contexte de recherche rigoureux sur l'analyse automatis\'ee de documents.

La dimension fullstack de ce poste est couverte par mon exp\'erience sur GoFind (Laravel, NestJS, Next.js, MongoDB, Docker) et SOSDOCTA (plateforme de t\'el\'em\'edecine en production), qui d\'emontrent une ma\^itrise end-to-end du d\'eveloppement d'applications web complexes.

Je suis disponible d\`es septembre 2026 pour cette alternance de 12 mois. Construire des syst\`emes agentic \`a impact m\'etier r\'eel chez Natixis est un objectif que j'aborde avec des r\'ealisations concr\`etes et une motivation forte. Je reste \`a votre disposition pour un entretien.

Je vous prie d'agr\'eer, Madame, Monsieur, l'expression de mes salutations distingu\'ees.
