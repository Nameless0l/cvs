# Exemples de CV et Lettres de Motivation

Ce dossier contient des exemples de candidatures générées pour différents types d'offres.

## Fichiers disponibles

| Fichier | Type d'offre | Entreprise |
|---------|--------------|------------|
| `cv_loic_mbassi_rd_electronique_valotech.tex` | R&D Électronique / Hardware | ValoTec |
| `lm_loic_mbassi_rd_electronique_valotech.tex` | LM R&D Électronique | ValoTec |
| `exemple_cv_embedded_easymile.tex` | Embarqué / Optimisation C | EasyMile |
| `exemple_cv_mlops_safran.md` | IA Embarquée / MLOps | Safran |

## Comment utiliser ces exemples

1. Identifier le type d'offre (Web, IA, Embarqué, FPGA, R&D...)
2. Consulter l'exemple le plus proche
3. Adapter le profil et les expériences selon l'offre spécifique
4. Vérifier les couleurs d'entreprise dans `copilot-instructions.md`

## Templates disponibles

- **Template 1 (article)** : Template principal avec FontAwesome, utilisé pour la plupart des candidatures
- **Template 2 (resume)** : Template compact style américain, nécessite `resume.cls`
